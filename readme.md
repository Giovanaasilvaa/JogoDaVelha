## 🎮 Jogo da Velha
Este é um **Jogo da Velha simples** desenvolvido com **HTML, CSS e JavaScript**. O projeto foi criado como parte de um **trabalho acadêmico na Fatec Campinas para a disciplina de Programação em Scripts**.

## 🛠️ Funcionalidades
1. Jogo para dois jogadores: Cada jogador escolhe seu símbolo ("X" ou "O").
2. Tabuleiro 3x3: Interface simples para interação.
3. Verificação automática de vencedor: O jogo detecta quando há um vencedor ou um empate.
4. Reinício da partida: Após o término, você pode reiniciar o jogo facilmente.

## 🧑‍💻 Tecnologias Utilizadas
1. HTML: Estruturação da página.
2. CSS: Estilização do jogo, tornando-o visualmente atrativo.
3. JavaScript: Lógica do jogo, incluindo controle das jogadas, verificação de vitória e reinício da partida.

## 🕹️ Como Jogar
- Jogador 1: Escolhe uma célula vazia no tabuleiro e clica nela para marcar com o símbolo "X".
- Jogador 2: Escolhe uma célula vazia e marca com o símbolo "O".
*O jogo continua até que um dos jogadores vença ou até que o tabuleiro esteja cheio, resultando em um empate.*
*Ao fim da partida, você pode clicar no botão de reiniciar para começar uma nova rodada.*
